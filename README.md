# Exploit Shop

Интернет-магазин цифровых товаров и лицензий с каталогом, категориями, страницами товаров, авторизацией, загрузкой изображений и административным управлением.

## О проекте

**Digital shop** — это fullstack-приложение для продажи цифровых товаров.  
Проект состоит из клиентской части на **Next.js** и серверной части на **NestJS** с базой данных **PostgreSQL**.

Сайт позволяет:
- просматривать категории и товары
- открывать отдельные страницы товаров
- работать с ценами, фичами и бейджами товаров
- загружать изображения
- регистрироваться и авторизовываться
- управлять товарами, категориями и ключами через backend
- использовать refresh/access auth через cookies

## Основной функционал

### Пользовательская часть
- просмотр категорий
- просмотр списка товаров
- страница отдельного товара
- изображения товара
- описание, цены, особенности и бейджи
- авторизация и регистрация

### Админ часть
- создание и редактирование категорий
- создание и редактирование товаров
- загрузка главного изображения и альбома
- управление ключами товара
- включение/скрытие товаров

### Backend возможности
- REST API
- JWT access/refresh токены
- cookies с `httpOnly`
- работа с PostgreSQL через TypeORM
- загрузка и раздача файлов из `public`
